Demo

This demo is built with ThreeJS. The information displayed here came from the Marvel API. And it is only used for demo purposes. This demonstration is not actually calling the Marvel API, instead it uses the JSON created for the presentation.

How to:

  • Select one or multiple characters.
  • Press a to accept the selection and show the Add Ons and their price.
  • Press b to cancel the selection and/or go back from the Add Ons list to the characters view.
  • Pres m to go full screen and f11 to cancel.
